De <code>div.wrapper</code> is aangepast en <code>main</code> heeft extra css-instellingen gekregen: 
``` css
.wrapper {
    ...
    display: flex;
    flex-direction: column;
}

main {
    ...
    flex: 1
}
```
Hierdoor staat de footer altijd onder aan de pagina
